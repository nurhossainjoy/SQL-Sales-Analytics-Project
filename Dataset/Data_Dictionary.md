# Data Dictionary

## Sales.xlsx

| Column | Data Type | Description |
|---------|-----------|-------------|
| spid | Text | Salesperson ID |
| geoid | Text | Geographic ID |
| pid | Text | Product ID |
| saledate | Date | Sales date |
| amount | Numeric | Sales revenue |
| customers | Integer | Number of customers |
| boxes | Integer | Number of boxes sold |
