# Sales Analytics Project using PostgreSQL

## Project Overview

This project demonstrates SQL-based sales analysis using PostgreSQL. It covers data exploration, table joins, Common Table Expressions (CTEs), window functions, aggregate functions, ranking, and profitability analysis.

The project analyzes sales performance across multiple regions, products, and salespersons using transactional sales data.

---

## Objectives

- Build a master sales table using multiple JOIN operations.
- Analyze yearly and monthly sales.
- Compare sales across countries and regions.
- Identify the top-performing salespersons.
- Calculate product-wise profit and profit margin.
- Practice advanced PostgreSQL concepts.

---
