# Results

This folder contains the outputs generated from the SQL queries performed in the Sales Analytics Project.

## Contents

### Query Output Screenshots

The screenshots demonstrate the results of the key SQL analyses performed in PostgreSQL using pgAdmin.

The analyses include:

- Data Exploration
- Master Sales Table Creation
- Yearly Sales Analysis
- Monthly Sales Trend
- Regional Sales Performance
- Top Salespersons by Location
- Product Profitability Analysis
- Window Function Examples (DENSE_RANK, CTEs)

## Purpose

These outputs showcase how SQL was used to transform raw transactional data into meaningful business insights.

The screenshots are included to provide a quick visual overview of the analysis without requiring users to execute the SQL scripts themselves.
